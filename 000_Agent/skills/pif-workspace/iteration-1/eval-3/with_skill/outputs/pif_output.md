# PIF 第十章：成分毒理資料
## 苯氧乙醇 (Phenoxyethanol) 安全評估

**評估情境：** 臉部留置型保濕乳霜，含量 0.8%，每次用量 0.5g，每日 1 次

---

## 1. 成分基本資料

| 項目 | 內容 |

| 中文名稱 | 苯氧乙醇 |
| INCI 名稱 | Phenoxyethanol |
| CAS No. | 122-99-6 |
| EC No. | 204-589-7 |
| 分子式 | C₈H₁₀O₂ |
| 分子量 | 138.16 g/mol |
| 功能 | 防腐劑 (Preservative) |
| **台灣法規上限** | **1.0%**（化粧品防腐劑成分基準） |
| 本配方含量 | **0.8%**（符合規定） |

---

## 2. 毒理學資料彙整

### 2.1 急性毒性 (Acute Toxicity)

| 測試項目 | 數值 | 物種/途徑 | 來源 |
|----------|------|----------|------|
| LD₅₀ | 1,260 mg/kg bw | 大鼠，口服 | ECHA 登錄資料 |
| LD₅₀ | > 2,000 mg/kg bw | 大鼠，皮膚 | ECHA 登錄資料 |
| LC₅₀ | > 1.97 mg/L/4h | 大鼠，吸入 | ECHA 登錄資料 |

**結論：** 急性毒性低，非高毒性物質。

### 2.2 皮膚刺激性 (Skin Irritation)

- OECD 404 兔子皮膚試驗：輕微刺激性
- 在 ≤ 1.0% 使用濃度下，皮膚刺激性可接受
- SCCS/1416/11：於上限濃度 1.0% 下，對正常皮膚無明顯刺激

### 2.3 眼部刺激性 (Eye Irritation)

- 對眼睛有中度刺激性
- 建議標示：「避免接觸眼睛」

### 2.4 皮膚致敏性 (Skin Sensitization)

- GPMT（豚鼠最大化試驗）：**陰性**
- LLNA（小鼠局部淋巴結分析）：EC₃ > 10%，非強效致敏物
- SCCS 評估：在 1.0% 使用濃度下，致敏風險可接受

### 2.5 基因毒性 (Genotoxicity)

| 測試 | 結果 |
|------|------|
| Ames 試驗 (OECD 471) | **陰性** |
| 染色體畸變試驗 (OECD 473) | **陰性** |
| 小鼠淋巴瘤試驗 (MLA) | **陰性** |

**結論：** 無基因毒性疑慮。

### 2.6 重複劑量毒性 (Repeated Dose Toxicity)

| 研究 | NOAEL | 物種/途徑 |
|------|-------|----------|
| 90 天口服 | **150 mg/kg bw/day** | 大鼠，口服 |
| 28 天皮膚 | 600 mg/kg bw/day | 大鼠，皮膚塗抹 |

> **選用 NOAEL = 150 mg/kg bw/day**（90 天大鼠口服，最保守數值）

### 2.7 生殖與發育毒性

高劑量（> 500 mg/kg bw/day）才出現輕微發育影響，與化粧品使用濃度相差極大。SCCS 未將其列為生殖毒性關注物質。

### 2.8 致癌性

- IARC：未分類
- EU CLP：未列為致癌物
- 無充分證據顯示具致癌性

## 3. 暴露量計算 (SED)

依 SCCS Notes of Guidance 第 12 版（SCCS/1647/22，2023）：

**公式：** `SED = (A × C × DAp) / BW`

| 參數 | 數值 | 說明 |
|------|------|------|
| A（每日接觸量） | 0.5 g/day | 每次 0.5g，每天 1 次 |
| C（配方濃度） | 0.008 | 0.8% = 0.008 |
| DAp（皮膚吸收率） | 2.3%（= 0.023） | SCCS/1416/11 人體皮膚體外滲透數據 |
| BW（體重） | 60 kg | SCCS 標準成人 |

```
每日成分接觸量 = 0.5 g × 0.008 = 0.004 g = 4 mg/day

SED = (4 mg/day × 0.023) ÷ 60 kg
    = 0.092 ÷ 60
    = 0.001533 mg/kg bw/day
```

**SED = 0.00153 mg/kg bw/day**

---

## 4. 安全臨界值 (MoS)

**公式：** `MoS = NOAEL / SED`

```
MoS = 150 mg/kg bw/day ÷ 0.00153 mg/kg bw/day
    ≈ 98,039
```

**MoS ≈ 98,000**

---

## 5. 安全評估結論

| 評估項目 | 結果 |
|----------|------|
| 台灣法規上限 | 1.0% |
| 本配方濃度 | 0.8%（✅ 符合） |
| NOAEL | 150 mg/kg bw/day |
| SED | 0.00153 mg/kg bw/day |
| **MoS 計算值** | **≈ 98,000** |
| SCCS 最低可接受門檻 | 100 |
| **符合安全標準** | **✅ 是（MoS 遠高於 100）** |
| 基因毒性 | ✅ 無 |
| 致癌性 | ✅ 無已知風險 |

**安全評估結論：**
苯氧乙醇以 0.8% 用於臉部留置型保濕乳霜，每日 1 次使用，MoS 約 **98,000**，遠超 SCCS 安全門檻 100。無基因毒性、無致癌性、致敏風險低，使用濃度符合台灣及 EU 法規規定。

**本成分在本評估條件下，對消費者不構成安全風險。評估結論：安全 (SAFE)。**

---

## 6. 其他安全注意事項

- 眼部刺激性：建議於產品標籤加註「避免接觸眼睛」
- 嬰幼兒（3 歲以下）：SCCS 評估指出新生兒皮膚吸收率可能較高，如產品用於嬰幼兒，建議重新計算 SED（體重調整）
- 職業暴露：美髮師等每日多次使用者，暴露量可能高於一般消費者，需另行評估
