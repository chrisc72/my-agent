# 毒理學安全評估報告
## 苯氧乙醇 (Phenoxyethanol)

**產品資訊**
- 產品類型：臉部留置型保濕乳霜
- 成分：苯氧乙醇 (Phenoxyethanol)
- 配方濃度：0.8%
- 每次用量：0.5 g
- 使用頻率：每日 1 次

---

## 1. 成分基本資料

| 項目 | 內容 |
|------|------|
| INCI 名稱 | Phenoxyethanol |
| CAS No. | 122-99-6 |
| 分子量 | 138.16 g/mol |
| 功能 | 防腐劑 |
| 台灣法規最高允許濃度 | 1.0% |

---

## 2. 毒理學資料

| 毒理終點 | 數值/結論 |
|----------|---------|
| 大鼠口服 LD₅₀ | 1,260 mg/kg bw |
| 皮膚刺激性 | 輕微（OECD 404） |
| 皮膚致敏性 | 無（GPMT 陰性） |
| 基因毒性 | 陰性（Ames, 染色體畸變） |
| NOAEL（90天大鼠口服） | 150 mg/kg bw/day |

---

## 3. SED 計算

```
每日接觸量 = 0.5 g × 0.8% = 4 mg/day
皮膚吸收率 = 2.3%（SCCS 數據）
SED = 4 mg × 0.023 / 60 kg = 0.00153 mg/kg bw/day
```

---

## 4. MoS 計算

```
MoS = NOAEL / SED = 150 / 0.00153 ≈ 98,000
```

MoS 遠高於安全門檻 100，評估結論：**安全 (SAFE)**
