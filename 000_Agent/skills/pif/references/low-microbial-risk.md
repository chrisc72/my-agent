# 微生物低風險性產品條件

來源：化粧品產品資訊檔案製作指引（衛生福利部食品藥物管理署）

用途：判斷產品是否符合「微生物低風險性」，若符合，第 12 項微生物檢測報告及第 13 項防腐效能試驗報告可由安全資料簽署人員評估後免除。

---

## 微生物低風險性產品條件（符合任一即可免除）

| 條件 | 說明 |
|------|------|
| **pH ≤ 3.0** | 酸性極強，微生物無法生長 |
| **pH ≥ 10.0** | 鹼性極強，微生物無法生長 |
| **無水配方（Anhydrous）** | 不含游離水分，例如：純油脂、蠟基產品、唇彩、粉末 |
| **含乙醇 ≥ 20%（體積比）** | 高濃度酒精具有效殺菌效果 |
| **充填溫度 ≥ 65°C** | 高溫充填可消除微生物，適用特定製程 |
| **水活性 (aw) ≤ 0.75** | 水活性過低，微生物無法繁殖 |
| **極性有機溶劑 > 10%** | 如丙二醇 (Propylene Glycol) > 10%，具抑菌效果 |
| **含鹼性化合物** | 例如：燙髮液、除毛膏等強鹼性製品 |
| **含氯化鋁 ≥ 25%（AlCl₃）** | 止汗劑類，高濃度鋁鹽抑菌 |
| **含過氧化氫 ≥ 3%（H₂O₂）** | 漂白劑、染髮劑等含高濃度雙氧水 |

---

## 免除申請方式

若產品符合上述任一條件，應由**安全資料簽署人員**於第 16 章「產品安全資料」中敘明：

1. 符合的免除條件（例如：「本產品含乙醇 40%，符合微生物低風險性條件」）
2. 免除項目（第 12 項和/或第 13 項）
3. 簽署人員姓名及日期

---

## 常見低風險性產品類型參考

| 產品類型 | 常見符合條件 |
|---------|------------|
| 古龍水、香水（高酒精） | 乙醇 ≥ 20% |
| 唇膏、口紅（無水） | 無水配方 |
| 眼影粉、腮紅粉（無水） | 無水配方 |
| 果酸煥膚產品（強酸） | pH ≤ 3.0 |
| 除毛膏（強鹼） | pH ≥ 10.0、含鹼性化合物 |
| 止汗劑 | 含氯化鋁 ≥ 25% |
| 漂髮粉（雙氧水成分） | H₂O₂ ≥ 3% |

---

## 注意事項

- 免除條件須有配方數據支持，例如附上 pH 量測報告或乙醇含量分析
- 混合產品（如含有水相且 pH 接近中性）通常不符合免除條件
- 眼部、嬰幼兒、黏膜部位用途產品，即使符合免除條件，建議仍進行微生物測試
- 免除決定由安全資料簽署人員負責，評估時應保守
